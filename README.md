## [emEYE简介](https://github.com/Qful/emEYE)

[emEYE](https://github.com/Qful/emEYE)项目主要用于监听环境中的无线电状态，可以说是一种通信类的传感器，其价值需要不断被挖掘，但随着物联网建设越来越多的无线电设备被部署，监听无线信道变得有意义。
